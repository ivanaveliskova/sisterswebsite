FCKeditor.Packager

This program compacts and optimizes the FCKeditor source files, found at the "editor/_source" directory, on fewer, smaller files. The new files will be created in the "editor/js" directory. Any existing file in the "js" directory will be overwritten.

FCKeditor.Packager is a free software for the FCKeditor community. It is based on CKPackager.

Usage Instructions

 FCKeditor.Packager [FCKeditor dir] [-h|/h|-?|/?]

[FCKeditor dir] - Optional. The directory where FCKeditor is installed. If omitted, the packager must run in that directory.

-h /h -? /? - Shows the command line help.

Requirements

FCKeditor.Packager runs on Windows operating systems with the Microsoft .Net Framework installed (almost default nowadays). It runs also on Linux systems with the Mono framework. Just type "mono CKPackager.exe".
